## Getting Started
### Prerequisites

First you need to have npm install in your system.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/kedarSedai/Todo
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the App
   ```JS
   npm start
   ```
5. Go to local host 3000
     ```sh
     http://localhost:3000/
   ```
